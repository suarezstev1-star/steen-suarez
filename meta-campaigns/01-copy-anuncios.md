# Copy de anuncios — Listos para copiar y pegar

Tres variantes completas. Elige una o rota las tres como A/B test.

Ninguna menciona dinero, comisiones, libertad financiera, seguros como
producto, tarjetas, préstamos, inversión, ni promesas de ingresos.

---

## Variante A — Enfoque carrera profesional

**Texto principal (Primary text):**
```
¿Buscas un cambio profesional en 2026?

Estamos contratando representantes bilingües (español/inglés) para unirse a
nuestro equipo. Ofrecemos:

• Capacitación completa desde el primer día
• Horario flexible
• Trabajo remoto disponible
• Crecimiento dentro del equipo
• Ambiente de trabajo colaborativo

No se requiere experiencia previa. Buscamos personas con actitud,
compromiso y ganas de aprender.

Aplica hoy y conversamos.
```

**Título (Headline):**
```
Estamos contratando · Aplica hoy
```

**Descripción (Description):**
```
Posiciones bilingües disponibles. Capacitación incluida.
```

**CTA:** `Aplicar ahora`

## Palabras y frases PROHIBIDAS (no usar en ningún copy)

Si usas cualquiera de estas, Meta volverá a rechazar el anuncio:

- "Libertad financiera"
- "Sé tu propio jefe" (ok en otros contextos, pero riesgoso aquí)
- "Gana $X al mes/semana"
- "Ingresos ilimitados"
- "Comisiones altas"
- "Cash", "money", "dinero" como gancho
- "Insurance", "seguros", "póliza"
- "Credit", "loan", "préstamo", "tarjeta"
- "Invierte", "inversión", "retorno"
- "Trabajo de 9 a 5" como algo negativo
- "Escapa de tu trabajo"
- "No más jefes"
- Emojis de dinero 💰💵💸🤑

## Palabras SEGURAS que sí puedes usar

- "Contratando", "hiring", "posiciones disponibles"
- "Carrera", "oportunidad profesional"
- "Capacitación", "entrenamiento", "training"
- "Equipo", "team"
- "Bilingüe"
- "Horario flexible", "remoto", "desde casa"
- "Aplicar", "aplicación", "postular"
- "Crecimiento profesional"
- "Sin experiencia previa"
