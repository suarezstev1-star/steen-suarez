# Pasos para arreglar la campaña en Meta Ads Manager

Sigue este orden exacto. No edites el anuncio rechazado: duplica y trabaja sobre la copia.

## 1. Duplicar la campaña rechazada

1. Entra a Ads Manager → pestaña **Campañas**
2. Selecciona la campaña `SaleADS.AI | VENTAS Instagram | +Advantage Prompt | IMG`
3. Click en **Duplicar** (icono de dos hojas)
4. Elige **Duplicación rápida**
5. Renombra la nueva a: `RECLUTAMIENTO | The Cash Team | Empleo | IMG`

## 2. Activar Categoría Especial de Anuncios (CRÍTICO)

1. Abre la campaña duplicada
2. Click en **Editar** (nivel campaña, no conjunto de anuncios)
3. Baja a **Categorías especiales de anuncios**
4. Activa el toggle y marca **"Empleo"** (Employment)
5. Confirma el país: **Estados Unidos** (o el país donde contrata Miguel)
6. Guarda

> Esto es obligatorio. Sin este paso, Meta seguirá clasificando el anuncio
> como financiero y lo rechazará de nuevo.

## 3. Revisar el Objetivo de campaña

Para reclutamiento usa uno de estos (NO uses "Ventas"):
- **Tráfico** → landing con formulario de aplicación
- **Interacción** → mensajes a WhatsApp / Messenger
- **Clientes potenciales (Leads)** → formulario instantáneo en Meta

Recomendado: **Clientes potenciales** con formulario instantáneo.

## 4. Configurar el conjunto de anuncios

1. Abre el conjunto de anuncios duplicado
2. **Ubicación:** Estados Unidos (o ciudades donde Miguel contrata)
   - Con Categoría Empleo, el radio mínimo es ~24 km
3. **Edad:** 18 - 65+ (no puedes restringir)
4. **Género:** Todos (no puedes restringir)
5. **Idiomas:** Español
6. **Segmentación detallada:** intereses como
   - "Búsqueda de empleo"
   - "Oportunidades de carrera"
   - "Trabajo desde casa" (si aplica)
   - NO uses intereses financieros (seguros, inversión, tarjetas)
7. **Ubicaciones del anuncio:** Automáticas

## 5. Reemplazar el anuncio

1. Abre el anuncio dentro del conjunto duplicado
2. **Identidad:** deja la página pero considera cambiar el nombre visible si dice "Insurance Agency"
3. **Texto principal / Título / Descripción:** usa los de `01-copy-anuncios.md`
4. **Creativo (imagen/video):** reemplaza por uno nuevo siguiendo `02-creativos-guia.md`
   - Sube una imagen nueva, NO reutilices la del anuncio rechazado
5. **CTA:** cambia a **"Aplicar ahora"** (Apply Now) o **"Más información"**
6. **URL de destino:** debe llevar a una página de aplicación/trabajo, NO a una de servicios financieros

## 6. Publicar

1. Antes de publicar, revisa el `04-checklist-compliance.md`
2. Click en **Publicar**
3. Espera revisión de Meta (24-48 horas)

## 7. Pausar/eliminar el anuncio rechazado original

1. Vuelve a la campaña original rechazada
2. Pásala a **Desactivada**
3. No la elimines todavía — mantenla como referencia por si necesitas apelar
