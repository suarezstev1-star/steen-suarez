# Campañas de Meta — Reclutamiento (Miguel / The Cash Team)

Este directorio contiene las plantillas corregidas y los pasos para arreglar
la campaña de reclutamiento que Meta rechazó por la Política de Prácticas
Discriminatorias (clasificada como servicio financiero).

## Estructura

- `00-pasos-para-arreglar.md` — Guía paso a paso en Meta Ads Manager
- `01-copy-anuncios.md` — Textos listos para copiar y pegar (primary text, headlines, descriptions)
- `02-creativos-guia.md` — Qué poner y qué NO poner en imágenes y videos
- `03-segmentacion.md` — Configuración correcta de audiencia con Categoría Empleo
- `04-checklist-compliance.md` — Lista de verificación antes de publicar
- `05-respuesta-revision-meta.md` — Texto para solicitar revisión si vuelven a rechazar

## Resumen del problema

**Anuncio rechazado:** SaleADS.AI | VENTAS Instagram | +Advantage Prompt | IMG
**Motivo Meta:** Promoción de servicios financieros/seguros sujeta a políticas
de prácticas discriminatorias.
**Causa real:** La campaña es de RECLUTAMIENTO pero no estaba declarada en
la Categoría Especial "Empleo", y el copy/creativos aludían a libertad
financiera y mencionaban "Insurance Agency".

## Solución en 3 pasos

1. Duplicar la campaña rechazada
2. Activar **Categoría Especial → Empleo** a nivel de campaña
3. Reemplazar copy y creativos con los de `01-copy-anuncios.md` y `02-creativos-guia.md`
