## 5. FORMATOS — ESTRUCTURAS

### Reel 45–60s (estructura por defecto)
1. **Hook (0–3s):** frase provocadora o pregunta técnica
2. **Problema (3–10s):** nombrar la frustración del agente
3. **Framework (10–40s):** 3 puntos claros con números/letras visibles en pantalla
4. **Demo visual (opcional 40–50s):** whiteboard, diagrama, o rol-play corto
5. **CTA suave (50–60s):** "En la miniclase gratis desarrollo esto — link en bio"

### Carousel 7–10 slides
1. **Slide 1:** hook + promesa de lo que vas a aprender
2. **Slides 2–3:** contexto del problema
3. **Slides 4–8:** framework / pasos / casos
4. **Slide 9:** recap visual
5. **Slide 10:** CTA + "guarda este post"

### Reglas visuales consistentes
- Fondo negro `#0D0D0D` o blanco `#FFFFFF`
- Texto en dorado `#FFD700` para emphasis
- Verde `#AAEE00` solo para acentos ("correcto", "sí")
- Rojo `#E63946` solo para errores ("no decir esto")
- Font: Arial o similar sans-serif bold
- Logo discreto en esquina inferior derecha
